Train Ticket Booking System(A console application)

Train Ticket Booking System (Java-Based Console Application) Developed a Train Ticket Booking System using core Java, designed to simulate real-world ticket reservation workflows including user authentication, train search, seat booking, and ticket management. The system follows a layered architecture with clear separation between entities, services, and utility classes. Data persistence is handled using JSON files, where user and train data are stored and managed using the Jackson library for serialization and deserialization. Users can sign up, log in securely (with hashed passwords), search for trains based on source and destination, view seat availability, and book or cancel tickets. The booking logic ensures seat validation and updates are reflected consistently in the data store.

Snake Game (Python OOP-Based Desktop Game)

Developed a classic Snake Game using Python, implementing core game mechanics such as movement, collision detection, score tracking, and dynamic object interaction. The application is built using Python’s built-in turtle graphics library, with a modular and object-oriented design. The logic is separated into multiple components: Snake → handles movement and body growth Food → generates random positions for food Scoreboard → tracks and displays score The game continuously updates the screen using a game loop, handling real-time keyboard input and ensuring smooth gameplay. Collision detection is implemented for both wall boundaries and self-collision, triggering game-over conditions.
